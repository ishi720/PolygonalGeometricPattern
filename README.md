# PolygonalGeometricPattern

多角形の辺上を移動する点から生成される幾何学的パターンを可視化するインタラクティブなWebアプリケーションです。

<img width="1200" height="800" alt="PolygonalGeometricPattern" src="https://github.com/user-attachments/assets/de6b6827-073f-47c3-be22-57a02bb4b7b1" />

## 概要

このプロジェクトは、正多角形の辺上を往復する点から線を引き、その交点が描く美しい軌跡をアニメーションで表現します。

## 技術スタック

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
